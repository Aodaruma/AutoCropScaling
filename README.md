# AutoCropScaling

同じ量の画像を自動的に切り抜き・拡大する AviUtl スクリプトです。

現在は **version 1.0** が最新です。

## AutoCropScaling.anm

以下はパラメーターの説明です。

- **トラックバー・チェックボックス**
  - scale%: 縦横において、もっとも幅が小さい部分の拡大率を設定します。
  - 縦横拡縮可: クリッピング・ピクセル拡張せず、画像を縦横にリサイズします。
